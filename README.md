# 금융 데이터 분석 학습

이 저장소는 Python과 Google Colab을 활용해 금융 데이터를 분석하고 관련 개념을 정리하기 위해 만들었습니다.  
금융 시계열 데이터의 기초 개념부터 수익률, 변동성, 시각화까지 단계적으로 학습하는 것을 목표로 합니다.

## 학습 내용

- 단순 수익률과 로그 수익률
- 장중 수익률
- 실현분산(Realized Variance)
- 실현변동성(Realized Volatility)
- 변동성 분석
- 금융 시계열 분석
- 금융 데이터 시각화

## 사용 도구

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels
- scikit-learn
- Google Colab
